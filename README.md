# Investment Calculator

A simple investment calculator built with React that helps users estimate potential returns based on initial investment, interest rate, and duration.

## Installation

Clone the repository and install dependencies:

```bash
git clone <your-repository-url>
cd <repository-name>
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```


## Technologies

- React
- Vite
- JavaScript
- CSS
